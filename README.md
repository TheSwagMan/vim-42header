# vim-custom-header
A great header for Vim.

Add your full name in `$FULLNAME` and your email in `$MAIL` variables, and you're good to go !

# Vundle
Add this into your `~/.vimrc` :

```vim
Plugin 'TheSwagMan/vim-custom-header'
```
